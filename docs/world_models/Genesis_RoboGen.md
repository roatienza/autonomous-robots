# 🤖 Genesis RoboGen
**Automatic Generation of Robotic Data**

**Links**: [GitHub](https://github.com/Genesis-Embodied-AI/RoboGen)

---

## 🎯 Core Objective
- **Data Synthesis**: Automatically generating high-quality robotic training data.
- **Curriculum Learning**: Creating a "curriculum" of tasks that gradually increase in difficulty.
- **Bridging the Gap**: Using world models to synthesize diverse scenarios that bridge simulation and reality.

---

## ⚙️ Key Technical Concepts
- **Automatic Labeling**: The world model not only generates the scene but also the ground-truth labels.
- **Scenario Diversity**: Synthesizing a wide range of object poses, lighting, and environments.
- **Feedback Loop**: Using agent performance to guide the generation of more challenging data.

---

## 🤖 Robotics Relevance
- **Scaling Robot Learning**: Overcoming the "data bottleneck" in robotics by generating millions of synthetic examples.
- **Robustness**: Training on a vast array of synthetic variations to make the robot more robust in the real world.
- **Zero-Shot Transfer**: Improving the likelihood of policies transferring from simulation to real hardware.
