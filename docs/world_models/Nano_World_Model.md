# Nano World Models: A Minimalist Implementation of Future Video Prediction

> **Link**: [GitHub](https://github.com/simchowitzlabpublic/nano-world-model) | [Project Page](https://simchowitzlabpublic.github.io/nano-world-model) | [arXiv](https://arxiv.org/abs/2605.23993)

## Overview
Nano World Models provide a minimalist, modular codebase for future video prediction centered around **diffusion forcing**. It aims to provide a compact and reproducible substrate for studying world-model design choices (architecture scale, action injection, etc.) across various environments including games and real-robot data.

## Key Contributions
- **Unified Interface**: Supports various generative objectives, model scales, and action-conditioning mechanisms.
- **Reproducibility**: Provides a standardized framework for evaluating autoregressive rollout behavior.
- **Experimental Substrate**: Enables controlled studies on how prediction parameterization and domain complexity affect video quality.
